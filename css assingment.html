CSS selectors

Dhruvi Padsala
·Jul 20, 2022·

2 min read

Edit

A CSS selector select the HTML elements you want to style. Selectors are part of CSS rule set. CSS selector select HTML elements according to its id, class, type, attribute etc.

We can divide CSS selector into few categories :

  • ID selector
  • Class selector
  • Universal selector
  • group selector

ID selector

The CSS id selector matches an element based on the value of the element's id attribute .In order for the element to be selected , its id attribute must match exactly the value given in the sector. The id selector is a unique identifier that links together content on the page . By appending that id value with hash symbol (#) to the section of the page . Since this can be used as part of the URL and for other scripting purpose, each id value on a page must be unique.

Class selector

The class selector select HTML elements with a specific class attribute. To select element with a specific class ,write a period (.) character, followed by the class name. you can also specify that only specific HTML elements should be a affected by a class. HTML elements can also refer to more the one class.

Universal selector

The universal selector select all HTML element on the page. By appending that universal value with dagger symbol (*) to the section of the page. HTML is also use as a wildcard element.

Group selector

The CSS grouping selector is used to select multiple element and style them together. this reduces the code and extra effort to declare common style for each element . To group selectors, each selector is separated by a space. the selector is a grouping method that select all the matching nodes. To reduce the size of style sheets, one can group selector in comma-separated lists.

pseudo elements

A CSS pseudo element is a keyboard added to a selector that lets you style a specific part of the selected elements, like: hover,after,before etc.

hover can be use to change a button's color when the user's pointer hovers over it.
